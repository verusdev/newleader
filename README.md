# Event CRM — SaaS для организаторов мероприятий

Мультитенантная CRM-система для организаторов мероприятий (свадьбы, юбилеи, выпускные, корпоративы). Интерфейс выполнен на **AdminLTE 3**.

## Возможности

- **Управление мероприятиями** — создание, планирование, статусы
- **Клиенты** — база клиентов с историей мероприятий
- **Гости** — списки гостей, подтверждение участия, категории
- **Задачи** — дедлайны, приоритеты, оценочная стоимость
- **Бюджет** — план/факт, статьи расходов, интеграция с подрядчиками
- **Подрядчики** — база контрагентов
- **Оплата подписки** — интеграция с ЮKassa

## Структура

- **Центральная часть** (`localhost`):
  - Лендинг с тарифами
  - Оформление подписки и оплата
  - Админ-панель управления тенантами

- **Tenant-часть** (на поддомене тенанта):
  - Панель управления (дашборд)
  - Мероприятия, клиенты, подрядчики
  - Гости, задачи, бюджет

## Установка

1. Установите зависимости:
   ```bash
   composer install
   ```

2. Настройте `.env`:
   ```
   APP_URL=http://localhost:8000
   DB_CONNECTION=sqlite

   YOOKASSA_SHOP_ID=ваш_shop_id
   YOOKASSA_SECRET_KEY=ваш_secret_key
   ```

3. Запустите миграции и сидеры:
   ```bash
   php artisan migrate
   php artisan db:seed --class=SubscriptionPlanSeeder
   ```

4. Запустите сервер:
   ```bash
   php artisan serve
   ```

5. Для работы tenant-доменов добавьте в `hosts`:
   ```
   127.0.0.1    myshop.localhost
   ```

## Поток регистрации

1. Пользователь заходит на `http://localhost:8000` (лендинг)
2. Выбирает тариф → оформляет подписку (имя, email, домен)
3. Оплачивает через ЮKassa
4. После оплаты автоматически создаётся тенант с собственной БД
5. Пользователь перенаправляется в свою CRM

## Технологии

- **Laravel 13**
- **stancl/tenancy** — мультитенантность
- **AdminLTE 3** — UI
- **yoomoney/yookassa-sdk-php** — платежи
